# 🛍️ Customer Shopping Behavior Analysis

## 📌 Project Overview
An end-to-end data analysis project examining 3,900+ customer transactions
to uncover spending patterns, customer segments, product preferences,
and subscription behavior using Python, SQL, and Power BI.

## 🛠️ Tools & Technologies
- **Python** (Pandas, NumPy) — Data cleaning & EDA
- **PostgreSQL** — Business queries & customer segmentation
- **Power BI** (DAX, Power Query) — Interactive dashboard
- **Excel** — Supporting analysis

## 📂 Project Files
| File | Description |
|------|-------------|
| `customer_shopping_behavior.ipynb` | Python EDA & data cleaning notebook |
| `customer_shopping_behavior.csv` | Raw dataset (3,900 rows, 18 columns) |
| `customer_behavior_dashboard.pbix` | Power BI interactive dashboard |

## 💡 Key Business Insights
- Male customers generated **2x more revenue** ($157,890) than female ($75,191)
- **Young Adults** contributed the highest revenue at $62,143
- Only **27% of customers** are subscribers — opportunity to grow retention
- **839 customers** used discounts but still spent above average purchase value
- Top-rated products: Gloves (3.86), Sandals (3.84), Boots (3.82)
- **80% of customers** are Loyal segment — strong repeat buying behavior

## 🔍 SQL Analysis Performed
10 business queries including revenue by gender, customer segmentation
(New/Returning/Loyal), discount dependency analysis, top products per
category, and age group revenue breakdown.

## 📈 Business Recommendations
- Promote exclusive subscriber benefits to convert the 73% non-subscribers
- Reward loyal customers to maintain retention
- Focus marketing spend on Young Adults and Express-shipping users
- Highlight top-rated products (Gloves, Sandals) in campaigns
